Taupuolis.lt saugumo statusas

📋 Kas vyksta?

GitHub Pages automatiškai sukonfigūruoja SSL sertifikatą custom domenams. Procesas gali užtrukti iki 24 valandų.

⏰ Kiek laiko užtruks?

Paprastai 1-24 valandos po domeno sukonfigūravimo. Kartais gali užtrukti iki 48 valandų.

🔧 Ką reikia padaryti?

Nieko! GitHub Pages viską padarys automatiškai. Jums tereikia palaukti.

📞 Kaip patikrinti?

Grįžkite čia po kelių valandų arba patikrinkite GitHub repository Settings → Pages.
